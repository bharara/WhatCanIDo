Requirement for running the project

1 - NetBeans with Java Web and tomcat
2 - XML-beans 2.3.0
3 - Commons-collections 4.4.1
4 - POI 3.7
5 - POI ooxml
6 - Microsoft Excel 2007 or later

Beware:
This project have 2 file paths hardcoded. They may need to be updated after moving the files
1 - GeneralFunctions FILE_NAME
2 - GeneralFunctions getCss

How to Start?

Go to index.html and run the file. The first time, server will take some time to load.
After that just go to http://localhost:8084/WhatCanIDo/
